# 一、什么原因导致的：

触发此错误的一些原因包括：
The Docker daemon is not running. Docker 守护程序未运行。Docker doesn’t shutdown cleanly. Docker 无法完全关闭。Lack of root privileges to start the docker service. 缺少启动 docker 服务的 root 权限。

解决方法：
（1）在终端中执行 – unmask docker.

如果 docker 被 masked 了，一般会有这样的提示：‘Failed to start docker.service: Unit is masked.’

（2）启动 start the docker daemon

（3）验证 docker 是否激活

如果出现以下内容，则已经激活了。
